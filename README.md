My dotfiles are organized in `stow`able form, and live in `~/.dotfiles`.

### Dependencies
-[stow](https://www.gnu.org/software/stow/stow.html)
-[vim-plug](https://github.com/junegunn/vim-plug)
-[tpm](https://github.com/tmux-plugins/tpm)
-[coc.vim](https://github.com/neoclide/coc.nvim)
..-node.js `curl -sL install-node.now.sh/lts | bash`
-[xst](https://github.com/gnotclub/xst)
-[xinit-xsession](https://aur.archlinux.org/xinit-xsession.git)
